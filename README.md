# FreshMart Retail Waste & Markdown Analytics

## Project Overview

This project analyses synthetic retail data for FreshMart, a fictional grocery retailer. The goal is to identify profit leakage caused by product waste, shrinkage, markdown timing, stock issues, and supplier delivery performance.

The project is built in Microsoft Excel using Power Query, PivotTables, data modelling, and dashboard design.

## Business Problem

FreshMart wants to reduce unnecessary waste, improve markdown recovery, monitor supplier performance, and identify stores or product categories causing avoidable profit loss.

## Dataset

The dataset is synthetic and created for portfolio and learning purposes. It contains separate raw data tables for:

- Products
- Stores
- Suppliers
- Sales
- Inventory
- Waste and shrinkage
- Markdowns
- Supplier deliveries

## Tools Used

- Microsoft Excel
- Power Query
- PivotTables
- Excel Data Model
- Dashboard design

## Project Workflow

1. Import raw Excel data using Power Query
2. Clean and transform each table
3. Create a structured data model
4. Build PivotTables and KPI calculations
5. Design an interactive Excel dashboard
6. Write business insights and recommendations

## Key Analysis Areas

- Sales and gross profit performance
- Waste and shrinkage loss
- Markdown sell-through and recovery
- Inventory expiry risk
- Supplier delivery performance
